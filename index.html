O que é

Avatar falante, sem abrir o HeyGen

VideosAvatar é o pacote INEMA que transforma um roteiro de texto num vídeo do seu avatar no HeyGen e entrega pronto no Telegram. Toda a chatice (chamar a API, esperar o render, baixar, medir o custo, mandar no bot) já vem resolvida em duas skills do Claude Code.

🎭 Seu avatar, sua voz

Usa o photo avatar do Nei (engine Avatar IV) com a voz clonada, forçada para português. Dá pra trocar por qualquer um dos avatares e das 46 vozes PT da conta.

⚡ Um comando

Você manda a fala; ele cria o vídeo, espera o render, baixa o MP4 em ~/projetos/output/heygen/ e entrega no bot do openpcbot com legenda do custo.

💳 Dois cofres de crédito

Escolha entre o wallet de API (pay-as-you-go) ou os créditos da assinatura (via MCP). Mesma geração, billing diferente — você decide qual queimar primeiro.

Como funciona

O pipeline, ponta a ponta

Do texto ao bot, sempre o mesmo caminho — muda só de onde sai o crédito.

Roteiro (texto)→ HeyGen: avatar + voz→ Render (polling)→ Download MP4→ Bot do openpcbot

🧍 Tipos de avatar no HeyGen

Avatares prontos: +1.100 públicos, é só escolher o ID.

Instant / Digital Twin (Avatar V): clona seu rosto e voz a partir de um selfie de ~15s. Melhor para vídeo.

Photo Avatar (Avatar IV): vira apresentador falante a partir de 1 foto. É o avatar do Nei.

Criar o avatar é feito no app do HeyGen; depois a automação só usa o avatar_id.

🔌 Dois caminhos de geração

API key → skill heygen-cli. Chama POST /v3/videos, gasta o pool de API (pay-as-you-go).

MCP remoto → skill heygen-mcp. Conector OAuth, gasta os créditos da assinatura.

Os dois entregam no mesmo bot e usam os mesmos IDs de avatar/voz. Detalhe de billing na seção Créditos.

Pré-requisitos

O que precisa estar no lugar

Tudo é lido do .env do openpcbot. Nenhuma credencial fica na página nem no repo.

🔑 Conta HeyGen com crédito

Um dos dois cofres: wallet de API (pay-as-you-go, a partir de US$5) ou assinatura com créditos de plano.

🗝️ Variáveis no .env

No .env do openpcbot:

# HeyGen
HEYGEN_API_KEY=…
HEYGEN_AVATAR_ID_NEI=…
HEYGEN_VOICE_ID_NEI=…
# entrega no bot
TELEGRAM_BOT_TOKEN=…
ALLOWED_CHAT_ID=…

⚙️ Runtime

Node 18+ (para o CLI). Para o caminho da assinatura, o conector MCP do HeyGen conectado via OAuth.

Guia de uso · passo a passo

Gerando o vídeo

No Claude Code, é só pedir em linguagem natural ("faz um vídeo do meu avatar falando X") — as skills disparam sozinhas. Por baixo, isto é o que roda.

1

Gerar pela API (skill heygen-cli)

O caminho padrão. Só a fala é obrigatória; o resto tem default (9:16, 720p, voz PT-BR, entrega no bot).

node ~/.claude/skills/heygen-cli/scripts/heygen.mjs --text "sua fala aqui"
2

Ajustar formato, voz ou avatar

Flags opcionais para mudar enquadramento, resolução, voz ou perfil — ou só salvar local sem mandar no bot.

node heygen.mjs --text-file roteiro.txt \
  --ratio 16:9 --res 1080p --voice <voice_id> --no-send
3

Descobrir vozes e avatares

Lista as vozes PT (46 na conta) e os perfis/avatares disponíveis.

node heygen.mjs voices --lang pt   # vozes em português
node heygen.mjs avatars          # seus avatares/perfis
node heygen.mjs --text "oi" --dry-run  # valida sem gastar (custo 0)
4

Gerar pela assinatura (skill heygen-mcp)

Para usar os créditos do plano em vez do wallet de API: conecte o MCP remoto do HeyGen (OAuth, sem API key) e autentique uma vez.

claude mcp add --transport http heygen https://mcp.heygen.com/mcp/v1/
# depois: /mcp → heygen → Authenticate (login no navegador)
5

Entregar no bot (caminho MCP)

Quando o MCP devolve a URL do MP4, o helper baixa, salva e manda no bot do openpcbot com a legenda.

node ~/.claude/skills/heygen-mcp/scripts/deliver.mjs \
  --url "<url_do_mp4>" --slug minha-campanha \
  --caption "🎬 via assinatura · 9:16 · PT-BR"
6

Conferir o custo

O CLI mede o custo real lendo a quota antes/depois e imprime quantos créditos saíram. Não estime — leia o número.

✓ custo: 51 créditos de API · restam 198
Créditos & custo

Dois cofres que não se cruzam

O HeyGen separa o billing da API do billing do plano web. A API key nunca toca os créditos da assinatura, e vice-versa. Por isso existem duas skills.

CaminhoCofre que gastaAuthSkillQuando usar

MCP remotocréditos da assinatura (plano)OAuthheygen-mcppreferido — créditos já pagos no mês

💡 Regra de bolso

Queime a assinatura primeiro (já está paga) e deixe o wallet de API como reserva.

📏 Avatar IV é a faixa cara

O avatar do Nei é photo avatar (Avatar IV). Por isso o default é 720p e vídeos curtos, mantendo o gasto baixo.

🇧🇷 Português é o ponto de risco

A voz clonada está cadastrada como inglês — a automação força locale pt-BR. Confira de ouvido o 1º render.
